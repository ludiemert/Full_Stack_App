### CRUD_FullStack
Full-stack project from start to finish => CRUD_FullStack - using CRUD Full Stack with Node, React &amp; MySQL

Development of a Full Stack CRUD for a user registry, where the backend will be in node, frontend in React.JS and MySQL as the database.

  #### Start Front-End: PS C:\Users\user\Downloads\CRUD_FullStack\front> yarn start
    http://localhost:3000/

#### Start Back-End: PS C:\Users\user\Downloads\CRUD_FullStack\api> yarn start
   http://localhost:8800/  

A CRUD_FullStack website developed with Node, React &amp; MySQL. William Lucas learnings.

#### User registration
- You can add user, email, telephone and date of birth to the system;
- Can remove complete user from the system;
- You cannot leave fields unfilled,
- Receive a confirmation message when creating, deleting or when a field is missing.
